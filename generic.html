Data Encryption Standard (DES) in PYTHON

In this project, I worked on implementing a simplified block cipher encryption algorithm. The expand function expands a 6-bit input to 8 bits based on a predefined table. The s1 and s2 functions apply S-box transformations to 4-bit inputs, outputting 3-bit values. The f function combines these steps: expanding input, XORing with a key, and applying S-boxes. The subkey function generates a round-specific subkey from the primary key. The round function processes a 12-bit input, applying the f function and swapping halves. The encrypt function performs multiple rounds (5 in this case) of the round function on the input plaintext using the provided key. The main block tests these functions for correctness.

key Decryption
in PYTHON

In this project, I worked on defining an Encryption class with static methods for a simple encryption and decryption algorithm. The encrypt method takes a 16-bit input and uses a hard-coded key (0x8D5E). It XORs the input with parts of the key in three rounds, swapping key halves each round, and returns the result as two bytes. The decrypt method reverses the process: it combines two bytes into a 16-bit input and swaps key halves and XORs to recover the original input. A hard-coded test ensures the process works as expected, demonstrating the encryption and decryption of a given input.

    RC4
    in PYTHON

    In this project, I worked on implementing the RC4 stream cipher for encryption and decryption. The RC4 technique encrypts the plaintext after generating a random key of the same length. Each character in the plaintext is XORed with a pseudo-random byte produced by RC4 as part of the encryption process, including a critical scheduling method and a pseudo-random generating technique. The ciphertext is the outcome. The original plaintext can be recovered by XORing the ciphertext and decrypting it with the same key. This illustrates the main application of symmetric encryption using the RC4 method.
